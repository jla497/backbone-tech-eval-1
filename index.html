What is backbone.js?

Backbone is a widely used node.js package providing separation of business logic, server/client logic, and the displaying of information. The main features of backbone driving webapps includes:

  • Models: Objects holding business information/logic
  • Collections: Ordered groups of models
  • Views: Displays and templating for organizing the webapp
  • Routers: URI management

Models

Models are considered the main focus of backbone and hold all the information for a webapp.

Usage

Models interact through javascript via http requests with the server.

  • GET: locally retrieves an attribute of a model
  • SET: locally sets the value for attributes of a model
  • FETCH: requests from the server a specific object to be store in the model
  • SAVE: requests the server to save the changes to model to the database

Collections

Collections include groups of models. Grouping models allows for easy manipulation and retrieval of multiple models. Most of the functions that a model uses is also shared with collections. These can be modified to retrieve and communicate over larger amounts of data.

Usage

  • GET: locally retrieves a model from a collection
  • SET: locally sets a group of models in a collection
  • FETCH: requests from the server an array of objects which are stored as models
  • SAVE: requests the server to save the changes to each model in the collection to the database

Collections act similarly to arrays and have a set of functions to add, remove and retrieve elements (models)

  • PUSH: adds a model to the end of the collection
  • POP: removes and returns the model at the end of the collection
  • SLICE: adds a model to the front of the collection
  • SHIFT: removes and returns the model at the front of the collection
